# 🧠 Day 18 – Brain Dump Action Planner Skill

## 📌 Overview
This project demonstrates how to transform unstructured notes, meeting transcripts, brainstorming sessions, and voice memos into a structured and interactive dashboard using AI. The generated output organizes information into summaries, action items, risks, blockers, open questions, and additional notes without inventing missing details.

## ✨ Features
- 📄 Automatic summary generation
- 💡 Key takeaways extraction
- ✅ Structured action item tracking
- ❓ Open questions identification
- ⚠️ Risks and blockers analysis
- 🔄 Conflict detection
- 📋 Additional supporting notes
- 📱 Responsive HTML dashboard
- 🎨 Modern dashboard-style interface inspired by productivity tools

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Claude AI Custom Skill

## 🚀 Workflow
1. Create the `brain-dump-action-planner` custom skill.
2. Paste the provided description and instructions.
3. Input messy notes, transcripts, or brainstorming content.
4. Generate the interactive HTML dashboard.
5. Review summaries, action items, risks, and open questions.
6. Save the HTML file and capture screenshots.
7. Upload the project files to GitHub.

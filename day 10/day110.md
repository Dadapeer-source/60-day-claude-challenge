## ✨ Features

- 🌗 **Dark / Light Mode Toggle** — Smooth theme switching with persistent accent system
- ⌨️ **Typing Animation** — Hero title cycles through AI/ML Engineer, Full Stack Developer, IoT Builder, Data Analyst
- 📜 **Scroll Reveal Animations** — Sections and cards fade up as you scroll
- 📊 **Animated Skill Bars** — Gradient-filled proficiency bars triggered on scroll
- 🃏 **Project Cards** — Each project with description, tech stack tags, and hover effects
- 🏆 **Certifications & Achievements** — Color-coded cards for certs, awards, and academic wins
- 📬 **Contact Form** — Mailto-based form that opens your mail client pre-filled
- 🔗 **Active Nav Highlighting** — Nav links highlight as you scroll through sections
- 📱 **Fully Mobile Responsive** — Hamburger menu, fluid grid, touch-friendly
- ⚡ **Zero Build Step** — Single `.html` file, no npm, no bundler, no dependencies to install

## 🚀 Deployment

### Option 1 — Netlify Drop (Fastest)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop `index.html`
3. Your site is live instantly ✅

### Option 2 — GitHub Pages
```bash
# 1. Create a new repo named: Dadapeer-source.github.io
# 2. Clone it
git clone https://github.com/Dadapeer-source/Dadapeer-source.github.io

# 3. Copy your file in and rename it
cp dadapeer-portfolio.html index.html

# 4. Push
git add .
git commit -m "🚀 Launch portfolio"
git push origin main
```
Your site will be live at `https://dadapeer-source.github.io`

### Option 3 — Any Static Host
Just upload `index.html` to Vercel, Render, Cloudflare Pages, or any static hosting provider.

## ✏️ Customization

All content is defined in JavaScript data objects at the bottom of `index.html` — easy to update without touching the HTML structure.

```javascript
// Update your skills
const skills = [
  { label: 'Python / ML', pct: 90 },
  ...
];

// Add a new project
const projects = [
  {
    name: 'Your Project Name',
    desc: 'One-line description of what it does.',
    stack: ['Tech1', 'Tech2'],
    color: '#818cf8',
    link: 'https://github.com/...',
  },
  ...
];

// Add a certification
const certs = [
  { title: 'Cert Name', org: 'Issuing body / description', type: 'Certification', color: '#fbbf24' },
  ...
];
```

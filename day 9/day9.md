# 🥗 NutriScope – Nutrition Analytics Dashboard

## 📌 Overview

NutriScope is a nutrition analytics dashboard designed to help users track their daily food intake, monitor macro and micronutrients, identify nutritional deficiencies, and receive personalized dietary recommendations.

The project was built as part of an AI-powered application development challenge focused on iterative product building using AI.

---

## 🚀 Features

### 👤 User Profile
- Age
- Gender
- Height
- Weight
- Activity Level
- Dietary Preference

### 🍽 Food Logging
- Add Food Items
- Quantity Tracking
- Unit Selection
- Editable Food Log
- Remove Entries

### 📊 Nutrition Tracking
Tracks:

- Calories
- Protein
- Carbohydrates
- Fat
- Fiber
- Iron
- Calcium
- Vitamin C
- Vitamin D
- Vitamin B12

### 📈 Dashboard Analytics
- Energy Progress
- Macro Distribution
- Nutrient Completion
- Deficiency Analysis
- Excess Nutrient Analysis

### 💡 Recommendations
- Food Additions
- Food Swaps
- Portion Adjustments
- Personalized Suggestions

# Day 25 – AI Shark Tank Simulator

## Project
Built an AI Shark Tank Simulator using HTML, CSS, and JavaScript.

## Features
- Startup pitch form
- Four AI investor personas
- Interactive Q&A
- Startup scoring system
- Investment recommendation
- Valuation and funding analysis
- Responsive dark-themed UI

## Learning
- Product evaluation
- Startup pitching
- Business model assessment
- Investor thinking
- Front-end application development



## Conclusion
This project demonstrates how AI can simulate startup investment decisions and provide structured feedback for founders.

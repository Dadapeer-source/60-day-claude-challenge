# 🚗 Day 17 – AI Vehicle Cost & Fuel Analysis Dashboard

## 📌 Overview
As part of my #60DaysOfAI challenge, I built an AI-powered Vehicle Cost & Fuel Analysis Dashboard using a CSV dataset and Claude AI. The dashboard transforms raw vehicle data into interactive visualizations and business insights.

## 🎯 Objective
- Analyze fuel costs across different fuel types.
- Compare CO₂ emissions and maintenance expenses.
- Calculate E85 fuel economics and break-even points.
- Visualize vehicle operating costs through interactive charts.

## 🛠️ Technologies Used
- Claude AI
- HTML5
- CSS3
- JavaScript
- SVG Charts
- CSV Data Analysis

## 📊 Dashboard Features
- KPI Cards for cost metrics
- Fuel Cost per Kilometer Comparison
- CO₂ Emission Analysis
- Vehicle Age vs Cost Trend
- E85 Score Gauge
- Monthly Cost Estimation
- Fuel Type Comparison Cards
- Responsive Dark-Themed Dashboard

## 🔍 Key Insights
- Compared Petrol, Diesel, CNG, EV, and E85 fuel types.
- Evaluated maintenance cost and running cost per kilometer.
- Calculated E85 break-even pricing and cost penalty.
- Generated interactive visualizations for better decision-making.

## 📚 What I Learned
- Processing structured CSV datasets using AI.
- Building interactive dashboards with HTML and SVG.
- Presenting business metrics through visual analytics.
- Converting raw data into actionable insights.

## ⚠️ Disclaimer
This dashboard is created for educational and learning purposes as part of the AI challenge. The calculations and insights depend on the provided dataset and should not be considered financial or operational advice.

# ⚽ Football Intelligence Hub

## 📌 Overview
Football Intelligence Hub is an AI-powered sports analytics project that combines tournament prediction, football knowledge assessment, and personality analysis into one interactive experience.

The system analyzes football-related data to generate evidence-based insights, quizzes, and personalized recommendations for users with different knowledge levels.

## 🚀 Features

- 🏆 FIFA World Cup 2026 prediction report
- 📊 Confidence-based tournament analysis
- 🌍 Dark horse team identification
- ⭐ Players to watch
- 🧠 Interactive Football IQ Quiz
- 📈 Football Awareness Score calculation
- 👤 Fan classification system
- 🔴 Messi vs Ronaldo personality compatibility analysis
- 🎯 Football personality archetype generation
- 💡 Personalized recommendations for players, clubs, national teams, and rivalries

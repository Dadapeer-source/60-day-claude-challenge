# Day 20 – Face Puzzle Game

## Overview

Built a browser-based Face Puzzle Game using HTML, CSS, and JavaScript.

The application captures the user's face using the webcam and converts it into an interactive sliding puzzle.

## Features

- Webcam access using getUserMedia()
- Take photo from camera
- 3×3, 4×4, and 5×5 puzzle modes
- Drag and drop puzzle pieces
- Mobile touch support
- Timer and move counter
- Correct piece detection
- Win screen
- LocalStorage leaderboard
- Responsive design

## Technologies

- HTML5
- CSS3
- JavaScript
- Canvas API
- LocalStorage API
- MediaDevices API

## Learning Outcomes

- Camera integration in browser
- Canvas image processing
- Puzzle generation algorithms
- Drag-and-drop interactions
- Mobile touch handling
- State management
- LocalStorage persistence

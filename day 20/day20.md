# Face Puzzle Game 📸🧩

A browser-based Face Puzzle Game built with HTML, CSS, and JavaScript that uses the device camera to capture a photo and transform it into an interactive sliding puzzle.

## Features

- 📷 Live webcam access using `getUserMedia()`
- 📸 Capture a photo directly from the browser
- 🧩 Multiple puzzle difficulties:
  - 3×3
  - 4×4
  - 5×5
- 🔀 Randomized puzzle generation
- 🖱️ Drag-and-drop puzzle interaction
- 📱 Mobile-friendly responsive design
- ⏱️ Live timer
- 🔢 Move counter
- ✅ Correct piece tracking
- 🔄 Retake photo and reshuffle options
- 🎨 Modern UI with responsive layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- WebRTC (`getUserMedia`)
- Local Browser Storage

## How to Run

1. Open the project in a modern browser.
2. Allow camera permission.
3. Capture your photo.
4. Select a difficulty level.
5. Start the puzzle and solve it.

> Camera access requires HTTPS or localhost in most browsers.

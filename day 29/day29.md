# Day 29 – Operation Lifeline: Supply Chain Crisis Lab

## Project Overview

Built an interactive supply chain crisis simulator using React, HTML, CSS, and JavaScript.

## Features

- Random company generation
- Random crisis events
- War Room decision engine
- Supplier negotiation simulator
- CEO leadership quiz
- AI investment strategy
- Executive dashboard
- Replay functionality

## Technologies

- HTML5
- CSS3
- JavaScript
- React (CDN)
- Babel

## Key Learnings

- State management with React useState
- Enterprise dashboard UI
- Decision-based simulations
- Business logic implementation
- Supply chain operations

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Open **index.html** in any modern web browser.
4. Start the interactive healthcare story.
5. Choose one of the available options after every chapter.
6. Complete all eight scenes.
7. Restart the story to explore the experience again.

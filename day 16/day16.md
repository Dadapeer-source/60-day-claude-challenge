# Day 16 – Build Your First Stock Research Skill

## Objective
Created a reusable Claude Custom Skill named `stock-fundamental-research` for analyzing listed companies using fundamental analysis.

## Features
- Financial statement analysis
- Valuation metrics (P/E, P/B, EV/EBITDA)
- Growth and profitability review
- Ownership trends
- Peer comparison
- Risk analysis
- Evidence-based reporting
- No buy/sell recommendations

## Tested On
- Infosys Ltd.

## Key Learnings
- Learned how to build reusable AI workflows using Claude Skills.
- Understood structured stock research methodology.
- Generated a comprehensive fundamental analysis report with supporting metrics and charts.

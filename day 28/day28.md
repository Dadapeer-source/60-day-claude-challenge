# Hospital Admission Readiness Simulator

An interactive web application that simulates hospital admission workflows and teaches healthcare operations through hands-on decision making.

## Features

- Patient Admission Setup
- Admission Readiness Analysis
- Prior Authorization Workflow
- Insurance Verification
- Bed Assignment
- Documentation Review
- Risk Tracking
- Care Coordination
- Timeline Progress
- Governance Snapshot
- Final Admission Decision

## Technologies

- HTML5
- Tailwind CSS
- Vanilla JavaScript

## How to Run

Open `index.html` in any modern browser.

## Learning Outcomes

- Hospital Admission Process
- Prior Authorization
- Insurance Verification
- Clinical Documentation
- Utilization Review
- Healthcare Operations
